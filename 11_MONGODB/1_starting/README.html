1. no cmder damos o comando mongo
  2. o comando "show dbs" mostra os bancos de dados que temos no computador
  3. o comando "use nome_banco" cria um banco e o torna ativo. Este banco só estará visível pelo "show dbs" quando ele tiver documents
  4. "db.nome_coleção.insertOne({chave1: valor1, chave2: valor2 })" cria um document em uma collection no banco ativo
    • caso a collection não exista ela será criada
    • a chave passada no método insertOne é sem aspas, já o valor pode ser, dependendo do tipo (strings recebem aspas, booleanos e outros não)
    • os valores podem ser de diversos tipos, conforme especificado aqui
  5. a partir da inserção de documents (dados) no banco ele já será exibido através do comando "show dbs"
  6. para recuperar um document usamos o comando "db.nome_coleção.findOne()"
    • em findOne podemos passar um objeto com os critérios para encontrarmos um dado específico
    • se houver mais de um ele retornará apenas um
