1. iniciou o projeto dando npm init -y
  2. instalou os pacotes express, express-handlebars mysql2 sequelize e nodemon
  3. criou o index.js na raiz
  4. depois dos pacotes instalados ele foi em package.json e criou o script start definindo o comando do nodemon
  5. criou as pastas do mvc, além de public, routes e db na raiz
  6. criou no mysql a partir do workbench o banco que vamos nos conectar
  7. criou db/conn.js que cria uma instância de sequelize e tenta fazer a conexão com o banco de dados com ela (de acordo com o estudado em 9_SEQUELIZE). Utilizarei promise para fazer essa conexão ou tratar seu erro. Ao final é exportada a instância
  8. em index ele vai importar express, express-handlebars, vai chamar o banco, vai definir handlebars como template engine, middleware para ler o urlencoded, definiu o static entre outras configs e escutou a porta 3000
