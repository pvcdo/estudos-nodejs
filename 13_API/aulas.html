  2. npm i express nodemon
  3. script start em package.json
  4. index.js = { const express ; const app ; app.use(express.urlencoded) ; app.use(express.json()) ; app.get('/', res.json({objJS})) ; app.listen }

Postman

  • criar collection
  • na collection clicamos em "add request"
    1. Rota get
      1. Rota /
      2. em index utilizaremos a rota get / que retorna um objeto com apenas um par atributo valor
      3. escolhemos um nome para a requisição, o verbo get e passamos a rota
      4. com as informações básicas configuradas e salvas podemos enviar a requisição
    2. Rota post
        3. Rota /createproduct
      1. em index criamos a rota post /createproduct que declara duas consts que vem do body da requisição, depois damos um console de cada uma dessas const
      2. escolhemos um nome para a requisição, o verbo post e passamos a rota
      3. escolhemos a opção de configurar o body, escolhemos o formato raw e escrevemos o objeto JSON com os parâmetros esperados pela API

Status da resposta

Podemos ver os status de requisição na documentação do mozilla.

Para utilizar status na api nós utilizamos o método .status(código) na resposta da rota, antes da utilização do método json que é a resposta propriamente dita.
