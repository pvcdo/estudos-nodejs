1. começou como cópia da pasta anterior
  2. em views/edituser ele criou uma div que vai encapsular o formulário de inserção de endereço
  3. esse formulário é um post que tem action /adress/create
  4. dentro desse form ele colou um input hidden com name="UserId" e value={{user.id}}
    • o name é case sensitive e deve ser escrito do mesmo jeito que é apresentado aqui (que no caso é o mesmo jeito que a coluna se apresenta no banco)
  5. ele também criou form-control e inputs para inserção da rua, número e cidade
  6. ele inseriu css para estilizar o espaço de inserção de endereço
  7. em index.js ele criou a rota post /adress/create, que recebe e trabalha os dados passados no body da requisição, cria um objeto de endereço com esses dados
  8. o objeto de endereço é passado como parâmetro para o método Address.create (que trabalhará com promise), criando portanto um registro na tabela de address do nosso banco
  9. caso Address.create seja fulfilled haverá o redirecionamento para /users/edit/${UserId}
